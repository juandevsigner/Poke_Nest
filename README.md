# Run development

1. Clone respository

2. Run

```
yarn install
```

3. You must install CLI

```
npm i -g @nestjs/cli
```

4. Run DB

```
docker-compose up -d
```

5. Build DB with data seed

```
http://localhost:3000/api/v2/seed
```

# Stack use

- MongoDB
- Nest
- Docker
